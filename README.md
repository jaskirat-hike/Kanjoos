Kanjoos
=======

A Windows Phone 8 expense tracker for the thrifty.  

Details
-------
Written in C# and XAML  
Uses SQLite  
Created for PEC hack prelims

Functionality
-------------
Add income and expenses  
Track monthly expenses  
Keep top spendings in check  
Visualise monthly spendings
